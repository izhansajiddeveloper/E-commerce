# 🛒 E-commerce Website

A simple, responsive e-commerce front-end website built using **HTML**, **Tailwind CSS**, and **JavaScript**.

## 🚀 Features
- Responsive design
- Product listings
- Add to cart with localStorage
- Checkout form
- Contact page with EmailJS
